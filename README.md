## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the backend server

```bash
npm run server
```

Start the frontend

```bash
npm run client
```
